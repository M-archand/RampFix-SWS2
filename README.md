## Features

Helps minimize ramp bugs on CS2 surf maps.

**This port adds:**
- Refactor of the original SwiftlyS2 port
- Migrated to SwiftlyS2 GameHooks (no manual signatures)
- Configurable thresholds via `config.toml`

> Requires [SwiftlyS2 >= 1.4.0-beta1](https://github.com/swiftly-solution/swiftlys2/releases)

## Installation

- Place the RampFix folder in your addons/swiftlys2/plugins folder.
- Config located at addons/swiftlys2/configs/plugins/RampFix/config.toml
