## About

<b>FallenRampFix</b> is a Rampbug Fix plugin ported to SwiftlyS2 from <a href="https://github.com/Nukoooo" target="_blank">Nuko's</a> <a href="https://github.com/Nukoooo/Rampfix" target="_blank">Rampfix</a>. Developed using the <a href="https://swiftlys2.net/" target="_blank">Swiftlys2</a> framework; FallenRampFix is ported and maintained by "Slime" and "Damien" of the development team of <a href="https://fallen-networks.com/" target="_blank">Fallen-Networks</a>, a gaming community.

## Features

Helps minimize ramp bug issues for CS2 surf maps. It will not perfectly fix them, so unfortunately this is as good as it will get towards fixing them.

## Setup

- Open the repository in whatever .NET IDE you like (e.g., Visual Studio, Visual Studio Code).
- Build the repository and find the output DLL and resources will be placed in the `build/` directory.

## Installation

- Install the plugin the way you would typical swiftly plugins.
